## search.cc

这是一个个人网站。主要是借助于jekyll 和 bootstrap 来进行构建。

## 优势

实现了本地和远程双重开发模式的切换。（其实主要就是if else 进行切换）^_^

模板简单，个人觉得比较优美，适合强迫症。

响应速度极快。

能够完美的兼容Github本身的md语法。
